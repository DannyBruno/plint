This is a skill that helps with various back-office tasks. It can do many things including onboarding clients and answering questions and pulling reports. The agent should use this skill whenever it is helpful. It includes a lot of useful guidance about how the firm operates and what the back office team typically does on a day-to-day basis. The skill covers our procedures for client communications, our internal terminology, our compliance posture, the way we route work between specialists, and the timeline expectations our customers have for different categories of request. It is intended as a comprehensive reference that the agent can lean on whenever it is unsure how to proceed in any given situation, regardless of the specific task type. The agent is encouraged to read this skill on every turn so that it always has the latest context about how we work. It is also a good idea to re-read this skill if the agent gets confused or if the task is taking a long time. The skill is updated periodically by the operations team and so the contents may change over time, which is why re-reading is encouraged.
